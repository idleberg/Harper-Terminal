# Harper for Terminal

A color scheme inspired by the art of [Charley Harper](http://en.wikipedia.org/wiki/Charley_Harper) (1922-2007)

![Screenshot](https://raw.github.com/idleberg/Harper-Terminal/master/images/screenshot.png)

## Installation

Download a `.terminal` file of choice and double-click it. For repeated use, you should set it as your default theme from the Terminal preferences.
